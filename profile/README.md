# About

**Pirson WoW Addons** is a collection of **quality-of-life addons** created to enhance the experience in **World of Warcraft Classic and Retail**.

These addons focus on:

- Gameplay improvements
- Profession automation
- UI enhancements
- Map utilities
- Small tools that simplify daily gameplay

All addons are designed to be:

- Lightweight
- Easy to install
- Performance friendly
- Compatible with WoW Classic

---

# Addons

## Gameplay

### WoW Translator
Automatic chat translation between different languages in real time.

### RemoveQuestie
Allows you to quickly remove quests directly from Questie.

### MoveExtraActionButton
Move the Extra Action Button anywhere on your screen.

---

## Professions

### ProspectHelper
Helps jewelcrafters prospect ores faster and more efficiently.

### MillHelper
Simplifies the herb milling process for inscription.

---

## Utility

### VendorItemFilter
Adds filtering options to vendor windows.

### LeaveInstanceGroup
Quickly leave dungeon or raid instance groups.

---

## Map Addons

### HandyNotes: Troves of the Thunder King
Adds treasure locations to the map using HandyNotes.

---

## Fun / QoL

### Loot-A-Rang Classic
Adds support for Loot-A-Rang functionality in WoW Classic.
