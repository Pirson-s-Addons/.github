# Pirson WoW Addons

Collection of quality-of-life addons for **World of Warcraft Classic and Retail**.

## Addons

### Gameplay

- **WoW Translator**  
  Automatic chat translation.

- **RemoveQuestie**  
  Remove quests quickly from Questie.

- **MoveExtraActionButton**  
  Move the extra action button anywhere.

### Professions

- **ProspectHelper**  
  Helps jewelcrafters prospect ores faster.

- **MillHelper**  
  Makes milling herbs easier.

### Utility

- **VendorItemFilter**  
  Filter vendor items.

- **LeaveInstanceGroup**  
  Quickly leave instance groups.

### Map Addons

- **HandyNotes: Troves of the Thunder King**

### Fun / QoL

- **Loot-A-Rang Classic**
